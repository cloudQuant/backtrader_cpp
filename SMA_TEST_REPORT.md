## 核心技术实现亮点

### Kahan求和算法实现
```cpp
// 直接计算方法中的Kahan求和
double sum = 0.0;
double compensation = 0.0;

for (size_t i = 0; i < period_; ++i) {
    double value = input->get(-static_cast<int>(i));
    
    // Kahan求和算法
    double y = value - compensation;
    double t = sum + y;
    compensation = (t - sum) - y;
    sum = t;
}
```

### 两种计算模式的智能选择
1. **增量模式**：
   - 适用于连续数据流
   - 计算效率高
   - 轻微精度损失在可接受范围内

2. **直接模式**：
   - 适用于需要最高精度的场景
   - 使用Kahan求和算法
   - 计算开销稍高但精度最优
